# M.A.X. Map Manager

## Description

A comprehensive tool for managing game content in M.A.X.: Mechanized Assault & Exploration.

This application enables players to archive, and install (restore) maps along with their associated save files,
making it easy to organize and preserve your M.A.X. gaming experience.
